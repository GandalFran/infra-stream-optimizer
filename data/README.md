# Data Directory

This directory contains the NYC Taxi & Limousine Commission (TLC) Trip Record Data used for experiments.

**Attribution:**
Data provided by the [New York City Taxi and Limousine Commission (TLC)](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page).

**Git LFS:**
Large files (*.parquet, *.jsonl) are tracked using Git LFS. Ensure you have Git LFS installed:
```bash
git lfs install
git lfs pull
```
